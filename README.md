## 🚀 Ferramentas Utilizadas:

* Intellij

* Java 17

* Spring Boot 3.3.5

  <br> 

## 🔶 Principais Métodos Disponíveis

 <br> 

🔹 GET
```bash
# Retorna todos os registros.
$ http://localhost:8080/fornecedor/v1
```

🔹 GET
```bash
# Retorna o registro pelo ID.
$ http://localhost:8080/fornecedor/v1/{💲id}
```

🔹 POST
```bash
# Salva um registro.
$ http://localhost:8080/fornecedor/v1
```

🔹 PATCH
```bash
# Atualiza um registro.
$ http://localhost:8080/fornecedor/v1/{💲id}
```

🔹 DELETE
```bash
# Deleta um registro.
$ http://localhost:8080/fornecedor/v1/{💲id}
```
